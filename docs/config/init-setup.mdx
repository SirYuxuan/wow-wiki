# 快速安装

首次安装前，建议先退出游戏和战网，避免配置文件正在占用，导致清理或导入不完整。

## 推荐方式

优先使用雨轩魔兽世界插件管理工具完成清理和导入，这种方式更省事，也更不容易漏步骤。

### 第一步：清理旧配置

1. 打开雨轩魔兽世界插件管理工具。
2. 勾选 `Cache`、`Fonts`、`Interface`、`WTF` 四个目录。
3. 如果你还想保留旧界面和角色配置，请先自行备份。
4. 确认无误后执行一键清理。

![清理工具界面](https://oss.yuxuan66.com/obsidian/20260403121514158.png)

### 第二步：生成一份干净配置

清理完成后，不要立刻覆盖旧角色数据，先让游戏生成一套全新的基础配置。

1. 启动游戏并登录任意角色。
2. 进入角色后稍等一会，让客户端自动生成新的 `WTF` 配置。
3. 从游戏菜单正常退出。
4. 不要直接强制关闭游戏，否则配置可能写入不完整。

### 第三步：使用分享码导入

1. 打开新手盒子。
2. 搜索分享码 `73927916`。
3. 进入快速安装页面。
4. 勾选全部安装项后开始安装。
5. 安装完成后重新进入游戏，检查布局、字体和插件是否正常加载。

## 手动方式

如果你不使用工具，也可以手动清理旧目录，再继续后面的导入流程。

1. 打开魔兽世界安装目录。
2. 删除 `Cache`、`Fonts`、`Interface`、`WTF` 四个目录。
3. 如果有保留需求，请先自行备份。
4. 删除完成后，按上面的“生成一份干净配置”和“使用分享码导入”继续操作。

![手动删除目录示意](https://oss.yuxuan66.com/obsidian/20260403122233760.png)

## 安装后建议

- 首次进入游戏后，先等待插件完成初始化，再调整细节设置。
- 如果界面位置、字体或布局异常，优先查看“常见问题”页面。
- 每次整合包更新后，建议先阅读更新记录中的兼容性说明。
