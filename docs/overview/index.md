# 整合包介绍

## 项目定位

雨轩魔兽整合包 Wiki 面向希望快速搭建稳定游戏界面的玩家，重点覆盖：

- 常用战斗信息展示
- 团队与副本提醒
- 背包、任务、地图整合
- 常用界面美化与效率增强

## 适用建议

| 项目     | 建议                                    |
| -------- | --------------------------------------- |
| 游戏版本 | 按整合包发布说明使用对应客户端版本      |
| 安装方式 | 建议全新目录安装，避免与旧插件冲突      |
| 更新策略 | 大版本更新前先备份 `WTF` 与 `Interface` |

## 核心特色

### 开箱即用

提供基础布局和常用插件组合，减少手动拼装时间。

### 可继续个性化

保留常见 UI 自定义空间，适合后续微调动作条、单位框体、团队框架与提示模块。

### 以稳定为优先

优先选择成熟、兼容性较好的插件组合，降低重复报错与冲突概率。

## 安装前建议

1. 备份旧角色配置。
2. 确认客户端语言和插件版本匹配。
3. 首次进入游戏时，不要同时叠加过多额外插件。
4. 每次更新后先检查更新记录中的兼容性说明。
