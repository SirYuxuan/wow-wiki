| `HandyNotes` | 地图标记 | 在地图上显示实用标记和信息点。 |
| `HandyNotes_Midnight` | 午夜地图资料包 | 为 HandyNotes 补充对应区域的地图标记数据。 |
| `LDCombatStats` | 旧版战斗统计 | 旧版 LightDamage 残留目录，提示用户删除旧文件夹。 |
| `LightDamage` | 轻量战斗统计 | 提供简洁的战斗数据统计功能。 |

| `Mapster` | 地图增强 | 增强世界地图与小地图显示体验。 |
| `MeetingStone` | 集合石 | 提供更方便的组队与队伍管理能力。 |
| `MeetingStoneEX` | 集合石扩展 | 为集合石补充扩展功能。 |
| `MoveAny` | 界面移动 | 允许自由移动和调整各种界面框体。 |
| `MRT` | 团队工具 | 提供团队管理、战斗辅助和首领安排等功能。 |
| `MurlokExport` | Murlok 数据导出 | 查看并导出 Murlok.io 的高层秘境或 PvP 参考数据。 |

| `NewBeeBox` | 新手盒子助手 | 用于新手盒子角色数据识别与同步。 |
| `Pawn` | 装等对比助手 | 对比装备收益并提示升级选择。 |
| `Plater` | 姓名板增强 | 为高级用户提供高度可定制的姓名板功能。 |
| `RoyMapGuide` | 全地图指引 | 提供地图中的 NPC 与路线标记。 |
| `Syndicator` | 数据索引库 | 为背包或物品类插件提供数据索引支持。 |
| `WarpDeplete` | 大秘境计时 | 提供大秘境路线与超时信息辅助。 |
| `WaypointUI` | 世界路点界面 | 直接在世界场景中显示路径点与定位信息。 |
| `WeiyuAnneMedia` | 媒体资源包 | 提供独立的 SharedMedia 材质与声音资源。 |
| `WorldQuestTracker` | 世界任务追踪 | 更直观地展示世界任务信息。 |
| `YuXuanSpecial` | 雨轩工具箱 | 提供地图辅助、界面增强、战斗辅助与法师冰霜相关功能。 |
