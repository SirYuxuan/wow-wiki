# 常见问题

<FAQAccordion>
   <FAQItem question="进入游戏后界面错位">
      <p><strong>可能原因：</strong> 编辑模式未导入。</p>
      <p><strong>处理方法：</strong></p>
      <ol>
         <li>
            输入 <code>/yxs</code> 打开雨轩工具箱界面，点击加载通用布局。
            <p>
               <img
                  src="https://oss.yuxuan66.com/obsidian/20260403120454703.png"
                  alt="雨轩工具箱加载通用布局界面"
               />
            </p>
         </li>
         <li>重新加载界面。</li>
      </ol>
   </FAQItem>

   <FAQItem question="如何恢复默认配置">
      <ol>
         <li>备份当前 <code>WTF</code>。</li>
         <li>删除对应角色或账号下的配置文件。</li>
         <li>重新进入游戏并按整合包说明导入默认设置。</li>
      </ol>
   </FAQItem>
</FAQAccordion>
